# Cómo usar esta carpeta con Claude Design

Esta carpeta es el "kit de marca" de **Goldem Chip's**. Tenla linkeada en Claude Design
y, al pedir cualquier diseño, indícale que use estos archivos.

## Qué hay aquí

| Archivo | Para qué sirve |
|---------|----------------|
| `goldem-chips-design-system.md` | **El cerebro.** Reglas, colores, fuentes, voz, estructura. Claude Design lo lee y lo obedece. |
| `goldem-tokens.css` | **Los estilos listos.** Variables y componentes para que la web salga on-brand sin pensar. |
| `Goldem_Chips_Sistema_de_Diseno.html` | El manual visual (para ti y diseñadores). |
| `logo-goldem-chips.png` | **Logotipo oficial** (mapa de bits, alta resolución). Para incrustar/mostrar en web y piezas. |
| `logo-goldem-chips.jpg` | Logotipo en JPG (fondo no transparente). Úsalo solo si necesitas JPG. |
| `logo-goldem-chips.pdf` | Logotipo vectorial en PDF. Para impresión. |
| `logo-goldem-chips.ai` | Logotipo editable (Adobe Illustrator). **Fuente original**: de aquí se exportan SVG/PNG transparente. |

> Mantén el logo en esta carpeta. Para web, lo ideal es una versión **.svg** o **.png con fondo
> transparente** en alta: expórtala desde `logo-goldem-chips.ai` y agrégala — la web se verá más limpia.
> (El `.png` y `.jpg` actuales tienen fondo claro, no transparente.)

## Cómo pedirle a Claude Design (copia y pega)

**Para la página completa:**
> Usando el sistema de diseño de esta carpeta (`goldem-chips-design-system.md` y `goldem-tokens.css`),
> diseña la landing page de Goldem Chip's. Respeta la paleta, las fuentes (Fraunces, Oswald, Inter),
> el tono de voz y la regla 60/30/10. Incluye estas secciones: hero con el eslogan "El verdadero sabor"
> y botón "Pídelas ya", nuestros sabores (papa, plátano, yuca), cómo las hacemos (a mano, pequeños lotes,
> ingredientes naturales), dónde encontrarnos y pie con redes. Fondo base color papel, acentos en oro con mesura.

**Para una sección suelta:**
> Siguiendo el sistema de diseño de la carpeta, diseña solo la sección "Nuestros sabores" como tres
> tarjetas (papa, plátano, yuca) con foto, nombre en Fraunces y un botón. Mantén el estilo artesanal y cálido.

**Para redes:**
> Con el sistema de la carpeta, crea 3 plantillas de post cuadradas (1080x1080) para Instagram:
> una de "nuevo sabor", una de "cómo las hacemos" y una de promoción. Usa la paleta y el tono de voz definidos.

## Consejos para que salga como lo quieres

- Sé concreto con **qué sección** y **qué contenido**; deja que la carpeta defina el **estilo**.
- Si algo sale frío o genérico, recuérdale: *"más cálido y artesanal, menos corporativo; usa más el papel y el cacao, el oro solo como acento"*.
- Pídele variaciones: *"dame 2 versiones del hero"* y quédate con la que más te lata.
- Cuando una pieza te guste, dile *"guárdala como referencia de estilo"* para mantener consistencia.
